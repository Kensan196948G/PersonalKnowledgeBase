# 印刷レイアウト（PrintLayout）

## 概要

ノートを印刷に適したレイアウトで表示・出力する機能。

## 機能詳細

### 基本機能

- 印刷プレビュー
- 印刷用スタイルシート適用
- 直接印刷（システムダイアログ）

### レイアウト調整

- 余白の設定
- フォントサイズの調整
- 改ページ位置の制御
- 画像サイズの最適化

### 印刷オプション

- カラー / モノクロ
- 背景色の印刷有無
- ヘッダー/フッターの追加

## ユースケース

- 紙での確認・校正
- 会議資料の配布
- オフラインでの参照用
