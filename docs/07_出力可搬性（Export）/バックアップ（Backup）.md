# バックアップ（Backup）

## 概要

全データを一括でバックアップし、データ消失を防ぐ機能。

## 機能詳細

### バックアップ対象

- 全ノート（本文＋メタデータ）
- 添付ファイル・画像
- 設定ファイル
- タグ・フォルダ構造

### バックアップ形式

- ZIP圧縮
- フォルダ構造維持
- 日時付きファイル名

### バックアップ方法

- 手動バックアップ（ワンクリック）
- 自動バックアップ（定期実行）
- 増分バックアップ（変更分のみ）

## ユースケース

- 定期的なデータ保護
- 別PCへの移行
- 障害からの復旧

## バックアップ設定（案）

- 保存先フォルダの指定
- 自動バックアップの間隔
- 保持する世代数
