# 共有リンク（ShareLink）

## 概要

特定のノートを一時的に共有するためのリンクを生成する機能。

## 機能詳細

### 基本機能

- 読み取り専用リンクの生成
- 有効期限の設定
- パスワード保護（オプション）

### 共有方式（案）

- 静的HTMLの生成＋ホスティング
- 一時的なURLの発行
- QRコードの生成

## ユースケース

- 特定のメモを他者と共有
- プレゼン資料の配布
- チームへの一時的な情報共有

## 注意事項

- 個人利用が前提のため、常時共有は想定外
- セキュリティを考慮した設計
- 共有後のデータ管理
