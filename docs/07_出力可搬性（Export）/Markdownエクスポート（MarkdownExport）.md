# Markdownエクスポート（MarkdownExport）

## 概要

ノートをMarkdown形式でエクスポートし、他のツールでも利用可能にする機能。

## 機能詳細

### 基本機能

- 単一ノートのエクスポート
- 複数ノートの一括エクスポート
- フォルダ単位でのエクスポート

### エクスポート内容

- 本文（Markdown形式）
- メタデータ（YAML Front Matter）
- 添付ファイル（相対パス）

### 出力形式（案）

```markdown
---
title: ノートタイトル
tags: [tag1, tag2]
created: 2024-01-15
updated: 2024-01-20
---

# ノートタイトル

本文...
```

## ユースケース

- 他のMarkdownエディタでの編集
- Git管理への移行
- 静的サイトジェネレータでの公開
