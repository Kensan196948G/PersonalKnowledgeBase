# トラブルシューティング

このディレクトリには、開発中に発生した問題の解決方法と、
ユーザー向けのトラブルシューティングガイドが保存されています。

## ガイド一覧

### ユーザー向けガイド
- `QUICK_START_RESET.md` - ブラウザリセットクイックガイド（10秒で解決）
- `BROWSER_STATE_RESET_GUIDE.md` - ブラウザ状態リセット詳細ガイド

### 開発者向けガイド
- `FOLDER_FILTERING_VERIFICATION_REPORT.md` - フォルダフィルタリング検証レポート
- `E2E_TEST_ERROR_REPORT.md` - E2Eテストエラーレポート
- `E2E_TEST_QUICK_FIX.md` - E2Eテストクイック修正
- `fix-note-display.md` - ノート表示修正ガイド
- `EDITOR_FIX_SUMMARY.md` - エディタ修正サマリー
- `DATABASE_STATUS_REPORT.md` - データベース状態レポート

## よくある問題

### フォルダフィルタリングが動作しない
→ `QUICK_START_RESET.md` を参照

### ノートが表示されない
→ `fix-note-display.md` を参照

### E2Eテストが失敗する
→ `E2E_TEST_QUICK_FIX.md` を参照
