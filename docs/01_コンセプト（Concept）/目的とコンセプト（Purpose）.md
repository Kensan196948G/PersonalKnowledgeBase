# 目的とコンセプト（Purpose）

## 概要

本システムは、OneNote や Notion のような「**メモ＋画像貼り付け**」を中核とした、**個人利用に特化したナレッジベースシステム**である。

## 前提条件

| 項目 | 想定 |
|------|------|
| 同時編集 | 不要 |
| 権限管理 | 不要 |
| マルチユーザー | 想定しない |
| 優先事項 | 個人の思考整理・知識蓄積 |

## 重視するポイント

- **作り切れる**：小さく始められ、完成させられる規模感
- **壊れない**：シンプルな構造で安定動作
- **育てられる**：後から機能を追加・拡張できる柔軟性

## このアイデアの目的

- 市販ツールの「広すぎる」機能群から解放される
- 自分の思考パターンに最適化されたツールを持つ
- データを常に自分の手元に置く安心感
