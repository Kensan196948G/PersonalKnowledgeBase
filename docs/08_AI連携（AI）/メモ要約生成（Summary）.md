# メモ要約生成（Summary）

## 概要

AIを活用して長いメモの要約を自動生成する機能。

## 機能詳細

### 基本機能

- メモ全体の要約生成
- 要点の箇条書き抽出
- 指定文字数での要約

### 要約の種類

| 種類 | 用途 |
|------|------|
| 概要（1-2文） | 一覧表示用 |
| サマリー（1段落） | 内容把握用 |
| 要点リスト | 復習・確認用 |

### 生成タイミング

- 手動実行（ボタン/コマンド）
- 保存時に自動生成
- 一覧表示時にオンデマンド

## ユースケース

- 長いメモの内容を素早く把握
- 一覧での概要表示
- 過去メモの振り返り

## 技術的検討

- LLM API（OpenAI, Anthropic など）
- ローカルLLM（Ollama など）
- トークン数・コストの管理
