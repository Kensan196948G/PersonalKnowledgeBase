# リッチテキスト入力（RichText）

## 概要

プレーンテキストではなく、書式付きテキストを入力・表示できる機能。

## 機能詳細

### 基本書式

- **太字**（Bold）
- *斜体*（Italic）
- ~~取り消し線~~（Strikethrough）
- 下線（Underline）
- `インラインコード`

### 入力方式の検討

| 方式 | メリット | デメリット |
|------|----------|------------|
| Markdown記法 | シンプル、学習コスト低 | 見た目と入力が異なる |
| WYSIWYG | 直感的 | 実装が複雑 |
| ハイブリッド | 両方の利点 | 切り替えが必要 |

## ユースケース

- 重要な単語を強調したい
- コード片をインラインで表示したい
- メモの可読性を上げたい
