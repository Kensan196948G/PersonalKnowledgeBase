# 箇条書きリスト（Lists）

## 概要

箇条書き（番号なし）と番号付きリストを作成できる機能。

## 機能詳細

### リストの種類

#### 箇条書き（Bullet List）
- 項目1
- 項目2
  - ネスト項目
  - ネスト項目

#### 番号付きリスト（Numbered List）
1. 手順1
2. 手順2
   1. サブ手順
   2. サブ手順

### 期待される動作

- Enter で新しい項目を追加
- Tab / Shift+Tab でネスト操作
- 自動採番

## ユースケース

- アイデアを列挙したい
- 手順を順番に記録したい
- 階層的な情報を整理したい
