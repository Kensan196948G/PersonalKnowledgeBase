# チェックリスト（Checklist）

## 概要

タスクやToDoを管理するためのチェックボックス付きリスト機能。

## 機能詳細

### 基本動作

- [ ] 未完了タスク
- [x] 完了タスク

### 期待される動作

- クリックでチェック状態を切り替え
- 完了項目の視覚的な区別（取り消し線など）
- ネスト対応

## ユースケース

- 買い物リスト
- プロジェクトのタスク管理
- 日々のルーチン確認

## 拡張アイデア

- 完了率の表示
- 完了日時の自動記録
- 未完了のみフィルタ表示

## 検討事項（将来）

- [ ] タスク期限の設定
- [ ] リマインダー連携
- [ ] 完了タスクの自動アーカイブ
