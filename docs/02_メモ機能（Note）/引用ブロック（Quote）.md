# 引用ブロック（Quote）

## 概要

他の文献や発言を引用する際に使用するブロック要素。

## 機能詳細

### 表示例

> これは引用文です。
> 複数行にわたることもあります。

### 期待される動作

- 左側にボーダーラインを表示
- 背景色による視覚的区別
- ネスト引用への対応

## ユースケース

- 書籍からの引用
- Web記事の重要部分の抜粋
- 過去の自分のメモからの引用
- 会話や発言の記録

## 拡張アイデア

- 引用元URL/出典の記載欄
- 引用日時の自動記録
