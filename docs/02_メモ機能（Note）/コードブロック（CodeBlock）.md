# コードブロック（CodeBlock）

## 概要

プログラムコードやコマンドを見やすく表示するためのブロック要素。

## 機能詳細

### 基本機能

```python
def hello():
    print("Hello, World!")
```

### 期待される動作

- シンタックスハイライト
- 言語指定
- コピーボタン
- 行番号表示（オプション）

### 対応言語（案）

- Python, JavaScript, TypeScript
- HTML, CSS
- SQL
- Bash / Shell
- JSON, YAML, Markdown

## ユースケース

- コードスニペットの保存
- コマンドのメモ
- 設定ファイルの記録
