# ピン留め（Pin）

## 概要

重要なメモを一覧の上部に固定表示する機能。

## 機能詳細

### 基本動作

- ピン留めしたメモは常に上部に表示
- ピン留め解除で通常の並び順に戻る
- ピン留め状態の視覚的表示（アイコンなど）

### 期待される動作

- ワンクリックでピン留め切り替え
- ピン留めメモ同士の並び順制御
- フォルダ/タグ絞り込み時もピン維持

## ユースケース

- よく参照するメモを上部に固定
- 作業中のメモをすぐアクセスできるように
- 重要な連絡先・設定情報の固定

## 拡張アイデア

- ピン留め数の上限設定
- ピン留めメモ専用エリア
- ピン留めの優先度（複数段階）
