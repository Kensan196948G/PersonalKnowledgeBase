# URL自動カード化（URLCard）

## 概要

URLを貼り付けた際に、自動的にリンク先の情報を取得してカード形式で表示する機能。

## 機能詳細

### 表示要素

- ページタイトル
- 説明文（meta description）
- サムネイル画像（OGP画像）
- ファビコン
- URL

### 期待される動作

- URL貼り付け時に自動取得
- 取得失敗時はプレーンリンクにフォールバック
- 手動でカード化解除可能

## ユースケース

- 参考リンクの視覚的な管理
- ブックマーク代わりの利用
- 後で読む記事のストック

## 技術的検討

- OGP/meta情報の取得方法
- キャッシュの扱い
- オフライン時の動作
