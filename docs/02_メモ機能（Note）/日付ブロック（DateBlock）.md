# 日付ブロック（DateBlock）

## 概要

メモ内に日付を簡単に挿入し、日時情報を管理しやすくする機能。

## 機能詳細

### 基本機能

- 「今日」ボタンで現在日付を挿入
- カレンダーピッカーでの日付選択
- 日付フォーマットの設定

### 表示形式（案）

- `2024-01-15`（ISO形式）
- `2024年1月15日`
- `1/15（月）`
- `今日`、`昨日`（相対表示）

## ユースケース

- 日記・ログの日付記録
- 締め切りの記載
- イベント日の記録
- 作業履歴の管理

## 拡張アイデア

- 時刻の追加
- 日付範囲の表現
- カレンダー連携
