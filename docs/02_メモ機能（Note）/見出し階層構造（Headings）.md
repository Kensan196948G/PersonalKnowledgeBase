# 見出し階層構造（Headings）

## 概要

メモ内で階層的な見出しを使用し、構造化された文書を作成できる機能。

## 機能詳細

### 見出しレベル

- H1：メモタイトル相当
- H2：大セクション
- H3：中セクション
- H4〜H6：詳細セクション

### 期待される動作

- 見出しによる目次自動生成
- 見出し間のジャンプ
- 折りたたみとの連携

## ユースケース

- 長いメモを構造化したい
- 目次から特定セクションに飛びたい
- 議事録やレポートを整理したい
