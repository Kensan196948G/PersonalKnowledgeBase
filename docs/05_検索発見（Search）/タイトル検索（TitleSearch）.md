# タイトル検索（TitleSearch）

## 概要

メモのタイトルを対象に検索を行う機能。

## 機能詳細

### 基本機能

- タイトルに対するキーワード検索
- 部分一致検索
- インクリメンタルサーチ（入力中に結果表示）

### 期待される動作

- 即座に結果を表示
- ヒット箇所のハイライト
- 結果の並べ替え

## ユースケース

- 特定のメモを素早く探す
- タイトルの一部を覚えているメモの検索
- コマンドパレット的な利用
