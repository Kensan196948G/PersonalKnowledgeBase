# 日付範囲検索（DateRangeSearch）

## 概要

日付範囲を指定してメモを検索する機能。

## 機能詳細

### 検索対象

- 作成日時
- 更新日時
- （将来）本文内の日付

### 範囲指定方法

- カレンダーピッカー（開始日〜終了日）
- プリセット（今日、今週、今月、今年）
- 相対指定（過去7日、過去30日）

### 期待される動作

- 日付範囲内のメモを一覧表示
- 他の検索条件との組み合わせ
- 範囲のクリア

## ユースケース

- 先月作成したメモを確認
- 今週更新したメモを見る
- 特定期間の振り返り
