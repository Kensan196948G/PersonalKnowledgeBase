# あいまい検索（FuzzySearch）

## 概要

タイプミスや表記揺れがあっても目的のメモを見つけられる検索機能。

## 機能詳細

### 対応するケース

- タイプミス：`Javscript` → `JavaScript`
- 表記揺れ：`データベース` → `DB`
- 省略形：`config` → `configuration`
- 語順違い：`search fuzzy` → `fuzzy search`

### 期待される動作

- 完全一致がなくても類似結果を表示
- 類似度スコアでソート
- 「もしかして」の提案

## ユースケース

- 正確なスペルが分からないとき
- うろ覚えのキーワードで検索
- 入力ミスの許容

## 技術的検討

- レーベンシュタイン距離
- n-gram
- あいまい検索ライブラリ（Fuse.js など）
