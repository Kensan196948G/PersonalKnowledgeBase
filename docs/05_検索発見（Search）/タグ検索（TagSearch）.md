# タグ検索（TagSearch）

## 概要

タグを使ってメモを絞り込む検索機能。

## 機能詳細

### 基本機能

- 単一タグでの絞り込み
- 複数タグでの絞り込み（AND/OR）
- タグ一覧からの選択

### 期待される動作

- タグクリックで即座に絞り込み
- 選択中のタグの表示
- タグ絞り込みの解除

## ユースケース

- プロジェクト別のメモを見たい
- 特定カテゴリのメモを抽出
- タグの組み合わせで精密な絞り込み

## 拡張アイデア

- タグの除外検索（NOT）
- タグ階層を考慮した検索
- 保存済みフィルター
