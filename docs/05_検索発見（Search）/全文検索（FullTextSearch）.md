# 全文検索（FullTextSearch）

## 概要

メモの本文全体を対象に検索を行う機能。

## 機能詳細

### 基本機能

- 本文に対するキーワード検索
- 複数キーワード検索
- 検索結果のプレビュー表示

### 期待される動作

- ヒットしたメモの一覧表示
- ヒット箇所の周辺テキスト表示
- 検索語のハイライト

## ユースケース

- 「あの内容を書いたメモはどれだっけ？」
- 特定の単語が含まれるメモを全て抽出
- 情報の発掘・再発見

## 技術的検討

- インデックス作成（検索速度向上）
- 日本語形態素解析
- 検索エンジンライブラリの選定
