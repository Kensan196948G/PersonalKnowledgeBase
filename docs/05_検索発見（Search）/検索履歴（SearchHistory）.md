# 検索履歴（SearchHistory）

## 概要

過去の検索クエリを記録し、再利用できるようにする機能。

## 機能詳細

### 基本機能

- 検索履歴の自動保存
- 履歴一覧の表示
- 履歴からの再検索

### 期待される動作

- 検索ボックスでの履歴サジェスト
- 履歴の削除（個別/一括）
- 履歴の保存件数制限

## ユースケース

- 同じ検索を繰り返す
- 以前の検索条件を思い出す
- 検索パターンの分析

## 拡張アイデア

- 検索クエリの保存（名前付き）
- よく使う検索のピン留め
- 検索結果の変化追跡
