# 保存済みフィルター（SavedFilter）

## 概要

よく使う検索条件をフィルターとして保存し、ワンクリックで適用する機能。

## 機能詳細

### 基本機能

- 現在の検索条件を保存
- 保存したフィルターに名前を付ける
- フィルター一覧からの適用

### 保存できる条件

- キーワード
- タグ条件
- 日付範囲
- ソート順
- AND/OR条件

## ユースケース

- 「仕事関連の今月更新メモ」を保存
- プロジェクト別のビュー作成
- 定型検索の効率化

## 拡張アイデア

- フィルターのショートカット
- フィルターの共有（エクスポート）
- 動的フィルター（「今週」など）
