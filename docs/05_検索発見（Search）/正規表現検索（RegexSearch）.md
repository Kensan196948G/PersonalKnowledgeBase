# 正規表現検索（RegexSearch）

## 概要

正規表現（Regular Expression）を使った高度なパターン検索機能。

## 機能詳細

### 基本機能

- 正規表現パターンでの検索
- 大文字小文字の区別設定
- マッチ結果のハイライト

### 使用例

| パターン | 説明 |
|---------|------|
| `\d{4}-\d{2}-\d{2}` | 日付形式（YYYY-MM-DD） |
| `https?://\S+` | URL |
| `TODO:.*` | TODOコメント |
| `[A-Z]{2,}` | 大文字のみの単語 |

## ユースケース

- 特定パターンの文字列を検索
- コードやIDの検索
- フォーマット違反の検出

## 注意事項

- パフォーマンスへの影響に注意
