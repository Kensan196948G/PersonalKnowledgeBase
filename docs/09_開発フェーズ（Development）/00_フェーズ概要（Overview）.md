# 開発フェーズ概要

## プロジェクト特性

| 項目 | 内容 |
|------|------|
| 規模 | 中規模（個人ツール、段階的拡張） |
| 複雑度 | 中（フロント＋バック＋DB＋将来AI連携） |
| 開発者 | 1人 |
| 期間 | 長期育成型 |
| 重視 | **作り切れる・壊れない・育てられる** |

## 開発方針

### 個人開発の特性

- 本番環境という概念なし（ローカル完結）
- 最新技術を積極的に採用
- 実験的な機能も気軽に試せる
- 自分のペースで段階的に育成

### 4フェーズ構成

```
Phase 1: MVP（書くことに集中）
    ↓
Phase 2: 整理機能
    ↓
Phase 3: 知識化
    ↓
Phase 4: AI連携
```

## フェーズ一覧

### Phase 1: MVP（書くことに集中）

**目標**: 最小限の機能でメモを書いて保存できる状態

- TipTapエディタ基本実装
- 画像貼り付け（Ctrl+V）
- SQLite保存機能
- メモ一覧・基本検索

### Phase 2: 整理機能

**目標**: 書いたメモを整理・管理できる状態

- タグ管理システム
- フォルダ構造
- 高度検索（AND/OR）
- お気に入り・ピン留め

### Phase 3: 知識化

**目標**: メモ間の関係性を活用できる状態

- ノート間リンク [[ノート名]]
- バックリンク表示
- 関連ノート提案
- アウトライナー機能

### Phase 4: AI連携

**目標**: AIによる知識活用・発見支援

- ベクトル検索
- 類似ノート検索
- AI要約・質問応答
- 自動タグ付け
