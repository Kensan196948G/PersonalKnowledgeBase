# アウトライナー（Outliner）

## 概要

階層的な箇条書きで思考を整理するアウトライン機能。

## 機能詳細

### 基本機能

- 無限ネスト可能な箇条書き
- 折りたたみ / 展開
- ドラッグで項目の移動
- インデント / アウトデント（Tab / Shift+Tab）

### 期待される動作

- Enter で新しい項目
- 項目の折りたたみ状態の保持
- フォーカスモード（特定項目以下のみ表示）

## ユースケース

- ブレインストーミング
- 文章の構成検討
- タスクの階層分解
- 学習内容の構造化

## 拡張アイデア

- 項目へのタグ付け
- 項目単位でのリンク
- 完了マーク（チェックボックス連携）
- Workflowy/Dynalist風の操作感
