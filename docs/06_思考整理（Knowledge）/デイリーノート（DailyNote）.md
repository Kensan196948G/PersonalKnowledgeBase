# デイリーノート（DailyNote）

## 概要

日付ごとに自動作成されるノートで、日々の記録やフリーライティングを行う機能。

## 機能詳細

### 基本機能

- 「今日」ボタンで今日のノートを開く/作成
- 日付形式のタイトル（例：2024-01-15）
- カレンダーからの日付選択

### 自動生成内容（テンプレート）

```markdown
# 2024-01-15（月）

## 今日のメモ

## やったこと

## 明日やること
```

### 期待される動作

- 存在しなければ自動作成
- 前後の日へのナビゲーション
- カレンダービューとの連携

## ユースケース

- 日記・ジャーナリング
- 作業ログ
- フリーライティング
- モーニングページ
