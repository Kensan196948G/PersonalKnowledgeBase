# バックリンク（Backlink）

## 概要

あるノートを参照している他のノート（被リンク）を一覧表示する機能。

## 機能詳細

### 基本機能

- 現在のノートを参照しているノートの一覧
- バックリンク元のコンテキスト表示
- バックリンクからの直接ジャンプ

### 表示内容

- リンク元ノートのタイトル
- リンクが含まれる周辺テキスト
- リンク数のカウント

## ユースケース

- このノートがどこから参照されているか確認
- 知識の関連性を発見
- 孤立したノートの特定
