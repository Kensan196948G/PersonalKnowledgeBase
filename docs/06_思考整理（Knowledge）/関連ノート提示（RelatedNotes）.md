# 関連ノート提示（RelatedNotes）

## 概要

共通のタグやキーワードに基づいて、関連するノートを自動的に提案する機能。

## 機能詳細

### 関連性の判定基準

- 共通タグ
- 類似キーワード
- 同一フォルダ
- リンク関係

### 表示方法

- サイドバーに「関連ノート」セクション
- ノート下部に提案表示
- 関連度スコアでソート

## ユースケース

- 「この話題について他に何を書いていたっけ？」
- 忘れていたノートの再発見
- アイデアの連想・展開

## 技術的検討

- TF-IDF による類似度計算
- タグの重み付け
- リンク関係の重要度
