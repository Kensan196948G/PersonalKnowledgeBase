# ブロック参照（BlockReference）

## 概要

ノート内の特定ブロック（段落、リスト項目など）を別のノートから参照・埋め込みする機能。

## 機能詳細

### 基本機能

- ブロックへの一意なID付与
- 他ノートからのブロック参照
- 参照元の変更時に参照先も更新

### 参照形式（案）

```markdown
![[ノート名#ブロックID]]
{{embed: ノート名#ブロックID}}
```

### 表示方法

- インライン展開（内容を表示）
- リンク形式（クリックでジャンプ）
- トランスクルージョン（埋め込み）

## ユースケース

- 定義の再利用
- 引用の一元管理
- 知識の構造化
