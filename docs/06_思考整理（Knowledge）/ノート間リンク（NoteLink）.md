# ノート間リンク（NoteLink）

## 概要

メモ同士をリンクで繋ぎ、知識のネットワークを構築する機能。

## 機能詳細

### 基本機能

- `[[ノート名]]` 形式でのリンク作成
- リンククリックで対象ノートへ移動
- 存在しないノートへのリンク（赤リンク）

### 期待される動作

- 入力時のノート名補完
- リンク先のプレビュー表示
- リンク切れの検出

## リンク形式（案）

```markdown
[[完全なノート名]]
[[ノート名|表示テキスト]]
[[フォルダ/ノート名]]
```

## ユースケース

- 関連する概念同士を繋ぐ
- 用語集・辞書の構築
- プロジェクト内の相互参照
