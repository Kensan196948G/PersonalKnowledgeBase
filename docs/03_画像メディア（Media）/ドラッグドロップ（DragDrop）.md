# ドラッグドロップ（DragDrop）

## 概要

ファイルやフォルダをドラッグ＆ドロップでメモに追加する機能。

## 機能詳細

### 対応するドロップ対象

- 画像ファイル → 埋め込み表示
- PDFファイル → 添付/サムネイル表示
- テキストファイル → 内容埋め込みまたはリンク
- その他ファイル → 添付ファイルとして保存

### 期待される動作

- ドロップゾーンの視覚的フィードバック
- ファイル種別に応じた自動処理
- 複数ファイル同時ドロップ対応

## ユースケース

- エクスプローラーからの画像追加
- ダウンロードしたPDFの添付
- 複数ファイルの一括追加
