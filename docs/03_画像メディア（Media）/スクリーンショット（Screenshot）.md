# スクリーンショット（Screenshot）

## 概要

スクリーンショットを撮影してすぐにメモに貼り付ける機能。

## 機能詳細

### 撮影方法（案）

- ショートカットキーで全画面キャプチャ
- 範囲選択キャプチャ
- ウィンドウ指定キャプチャ
- 遅延キャプチャ

### 期待される動作

- 撮影後自動でメモに挿入
- タイムスタンプ付きファイル名
- 即時編集（トリミング等）

## ユースケース

- エラー画面の記録
- 手順書作成
- UIデザインの記録
- バグ報告用の証拠

## 技術的検討

- OS機能との連携
- 専用キャプチャツールの組み込み
- クリップボード監視
