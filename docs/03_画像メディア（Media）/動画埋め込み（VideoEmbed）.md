# 動画埋め込み（VideoEmbed）

## 概要

動画リンクをメモに埋め込み、プレビュー表示する機能。

## 機能詳細

### 対応サービス（案）

- YouTube
- Vimeo
- ローカル動画ファイル

### 表示要素

- サムネイル画像
- 動画タイトル
- 再生時間
- 再生ボタン

### 期待される動作

- URL貼り付けで自動認識
- クリックで再生（埋め込みプレイヤー）
- サムネイルのみ表示モード

## ユースケース

- 学習動画のブックマーク
- チュートリアルのまとめ
- 参考動画の整理
