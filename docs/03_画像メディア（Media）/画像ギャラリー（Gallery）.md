# 画像ギャラリー（Gallery）

## 概要

複数の画像をギャラリー形式で表示する機能。

## 機能詳細

### 表示レイアウト

- グリッド表示（2列、3列、4列）
- カルーセル表示
- マソンリー表示

### 期待される動作

- 複数画像の一括配置
- クリックでライトボックス表示
- 画像間のナビゲーション
- スライドショー再生

## ユースケース

- 旅行写真のまとめ
- 製品比較画像
- 作業工程の写真記録
- スクリーンショット集

## 拡張アイデア

- 画像のドラッグ並び替え
- ギャラリーへのキャプション
- 自動レイアウト調整
