# PDF添付（PDFAttachment）

## 概要

PDFファイルをメモに添付し、サムネイルプレビューを表示する機能。

## 機能詳細

### 基本機能

- PDFファイルの添付
- 1ページ目のサムネイル表示
- クリックで外部ビューアー起動

### 表示要素

- サムネイル画像
- ファイル名
- ページ数
- ファイルサイズ

## ユースケース

- 資料・論文の添付
- マニュアルの保存
- 請求書・領収書の管理

## 拡張アイデア

- PDF内テキスト検索
- 複数ページプレビュー
- 内蔵PDFビューアー
- ページ指定リンク
