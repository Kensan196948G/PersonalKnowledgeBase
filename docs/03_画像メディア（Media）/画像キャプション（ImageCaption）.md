# 画像キャプション（ImageCaption）

## 概要

画像に説明文（キャプション）を付与する機能。

## 機能詳細

### 基本機能

- 画像下部にキャプションテキストを表示
- キャプションの編集・削除
- 任意設定（空でもOK）

### 表示スタイル

- センター揃え
- 小さめフォント
- 画像と視覚的に関連付け

## ユースケース

- スクリーンショットの説明追加
- 図表の補足説明
- 出典情報の記載

## 拡張アイデア

- 画像番号の自動付与（図1、図2...）
- キャプション検索対応
- alt テキストとしての活用
