# 画像貼り付け（ImagePaste）

## 概要

Ctrl+V やドラッグ＆ドロップで画像をメモに簡単に貼り付けられる機能。

## 機能詳細

### 対応する貼り付け方法

- クリップボードからの貼り付け（Ctrl+V）
- ファイルのドラッグ＆ドロップ
- ファイル選択ダイアログ

### 対応フォーマット（案）

- PNG
- JPEG / JPG
- GIF
- WebP
- SVG

### 期待される動作

- 貼り付け即時プレビュー表示
- 自動ファイル保存（指定フォルダへ）
- 一意なファイル名生成

## ユースケース

- スクリーンショットの即時保存
- 参考画像のストック
- 作業記録の視覚的な保存
