# 画像サイズ切替（ImageResize）

## 概要

埋め込み画像の表示サイズを切り替えられる機能。

## 機能詳細

### サイズプリセット

| サイズ | 用途 |
|--------|------|
| S（小） | サムネイル・アイコン的表示 |
| M（中） | 標準表示 |
| L（大） | 詳細確認用 |
| 原寸 | 元の解像度で表示 |

### 期待される動作

- ワンクリックでサイズ切り替え
- サイズ設定の永続化
- レスポンシブ対応

## ユースケース

- 複数画像を並べて比較
- 詳細を確認したいときだけ拡大
- メモのスペース効率化

## 拡張アイデア

- カスタムサイズ指定
- 画像のトリミング
- 回転機能
