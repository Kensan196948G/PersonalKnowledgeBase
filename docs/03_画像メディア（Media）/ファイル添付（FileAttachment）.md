# ファイル添付（FileAttachment）

## 概要

任意のファイルをメモに添付して管理する機能。

## 機能詳細

### 対応ファイル

- ドキュメント（.docx, .xlsx, .pptx など）
- アーカイブ（.zip, .tar.gz など）
- その他任意のファイル

### 表示要素

- ファイルアイコン（拡張子別）
- ファイル名
- ファイルサイズ
- 添付日時

### 期待される動作

- ドラッグ＆ドロップで添付
- クリックでダウンロード/開く
- 添付ファイルの削除

## ユースケース

- 作業ファイルの一元管理
- プロジェクト関連資料の添付
- バックアップ用途
