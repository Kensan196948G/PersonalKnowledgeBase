# 画像補足メモ（ImageAnnotation）

## 概要

画像に対して補足情報やメモを付加する機能。

## 機能詳細

### 基本機能

- 画像に関連するテキストメモを追加
- 画像とメモをセットで管理
- 折りたたみ表示対応

### キャプションとの違い

| 項目 | キャプション | 補足メモ |
|------|-------------|----------|
| 長さ | 短い（1行程度） | 長くてもOK |
| 用途 | タイトル・説明 | 詳細メモ・考察 |
| 表示 | 常に表示 | 折りたたみ可 |

## ユースケース

- 画像の詳細な分析メモ
- スクリーンショットの操作手順
- 参考画像に対する自分の考え
