# テンプレート（Template）

## 概要

よく使うメモの形式をテンプレートとして保存し、新規メモ作成時に利用する機能。

## 機能詳細

### 基本機能

- テンプレートの作成・編集・削除
- テンプレートからのメモ新規作成
- テンプレート一覧の管理

### テンプレート例

#### 議事録テンプレート
```markdown
# 会議名

## 基本情報
- 日時：
- 参加者：
- 場所：

## アジェンダ
1.
2.

## 議事内容

## 決定事項

## TODO
- [ ]
```

#### 日記テンプレート
```markdown
# {{date}} の日記

## 今日やったこと

## 学んだこと

## 明日やること
```

## ユースケース

- 議事録の標準化
- 日記の習慣化
- レシピの定型フォーマット
- プロジェクト計画書
