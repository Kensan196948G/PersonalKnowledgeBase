# フォルダノートブック（Folder）

## 概要

メモをフォルダ（ノートブック）で階層的に整理する機能。

## 機能詳細

### 基本機能

- フォルダの作成・削除・リネーム
- メモのフォルダ間移動
- フォルダのネスト（階層化）

### 構造イメージ

```
📁 仕事
  📁 プロジェクトA
    📝 会議メモ
    📝 タスク一覧
  📁 プロジェクトB
📁 プライベート
  📝 買い物リスト
```

### 期待される動作

- ドラッグでの移動
- フォルダの展開/折りたたみ
- フォルダ単位でのエクスポート

## ユースケース

- プロジェクト別の整理
- カテゴリによる分類
- ライフ/ワークの分離
