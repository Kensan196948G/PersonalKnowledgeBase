# クイックスタート: ブラウザリセット

フォルダフィルタリングが正しく動作しない場合の簡単な解決方法です。

---

## 最速の解決方法（10秒）

1. ブラウザで以下のURLを開く:
   ```
   http://192.168.0.187:5173/reset-browser-state.html
   ```

2. **「すべてクリア」**ボタンをクリック

3. 確認ダイアログで**「OK」**

4. 自動的にホーム画面にリダイレクトされます

5. **2025年⑫フォルダ**をクリックして確認
   - **期待結果**: 30件のノートが表示される

---

## 検証が必要な場合

バックエンドAPIが正しく動作しているか確認したい場合:

```bash
./scripts/verify-folder-filtering.sh
```

期待される出力:
```
✅ 成功: データベース（30件）とAPI（30件）が一致
```

## トラブル時の連絡先

問題が解決しない場合は、以下の情報を添えてご連絡ください:

1. ブラウザのコンソールログ（F12 > Console）
2. ネットワークタブのエラー（F12 > Network）
3. リセット前と後のノート件数
